# Bubble-Sort
- 배열의 두 수를 선택한 뒤, 만약 그 두 수가 정렬되었다면 놔두고 아니라면 두 수를 바꾸는 방식으로 진행된다.

# 버블정렬의 방법
 
 초기상태 : [3,1,2,5,4]
 
1-1. [1,3,2,5,4]
- 3을 1과 비교하여 더 큰 숫자(3)이 뒤로 물러난다.

1-2. [1,2,3,5,4]
- 3을 2와 비교하여 더 큰 숫자(3)이 뒤로 물러난다.

1-3. [1,2,3,5,4]
- 3을 5와 비교하여 더 큰 숫자(5)가 자기 자리로 돌아온다.

1-4. [1,2,3,4,5]
- 5를 4와 비교하여 더 큰 숫자(5)가 뒤로 물러난다.
